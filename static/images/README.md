# Logo Location

**Place your logo.png file here:** `static/images/logo.png`

This is where the TTCPWorldwide logo should be placed. All HTML files are configured to reference:
```
/static/images/logo.png
```

The logo image should be:
- Filename: `logo.png`
- Format: PNG with transparent background
- Recommended size: 100px x 100px (will scale responsively)

## Files that reference this logo:
1. `templates/signup.html` - Sign up page
2. `templates/signin.html` - Sign in page  
3. `templates/index.html` - Home page
4. `templates/track.html` - Tracking page
